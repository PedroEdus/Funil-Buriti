# Painel CRM - Funil Digital

Aplicação em Streamlit para análise de leads, funil de vendas, cidades e origem de cadastro.

---

## 📊 Funcionalidades

- Visualização do funil de vendas
- Card de leads "Outros"
- Análise por cidade
- Análise por forma de cadastro
- Métricas de conversão
